# install-slims

## Step 1
cloning repository

```
git clone https://github.com/gibranzizzami/slims.git
```

## Step 2
Go to the slims directory

```
cd slims/
```

## Step 3
Change access for file script

```
chmod +x slims-shellscript.sh
```

## Step 4 
Running the installation

```
./slims-shellscript.sh
```

## Step 5
access slims on your browser
http://localhost/slims

notes
| Database information | Answer     |
| -------------------- | ------     |
| Database host        | localhost  |
| Database port        | 3306       |
| Database name        | slims      |
| Database username    | slims_user |
| Database password    | userslims  |
